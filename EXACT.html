Exact Algorithm for Connected Dominating Set (CDS) Computation

Objective:The Exact algorithm computes the Connected Dominating Set (CDS) of a graph by iteratively solving a maximum flow problem combined with binary search. It exploits the fundamental relationship between maximum flow and minimum cut in flow networks to accurately identify the minimal CDS. In this approach, the graph structure is carefully transformed into a flow network where clique structures and node degrees influence the edge capacities. By setting up the problem as a sequence of flow computations with gradually adjusted thresholds, the algorithm can systematically search for the smallest CDS that satisfies the connectivity and domination requirements.
